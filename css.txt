CSS Colors 
1. Color Names
2. RGB  / RGBA
3. HEX
4. HSL HSLA

RGB
Red, Green, Blue
RGB(0,0,0) = Black
RGB(255,255,255) = White

RGBA 
Red, Green, Blue, Alpha
Alpha= 0-1
RGBA(0,0,0,0.5) = Black with 50% transparency



CSS HEX color

-> Six-digit code
-> Represents the red, green, and blue 

# RRGGBB
1. # FF0000 = Red
2. # 00FF00 = Green
3. # 0000FF = Blue

 HEX color code is a six-digit code
 1. #000000 = Black
 2. #FFFFFF = White

#RGB



HSL

Hue, Saturation, Lightness
HSL(0, 100%, 50%) = Red

1. Hue: 
Color value  0 to 360

2. Saturation:
0% to 100%

3. Lightness:
0% to 100%

HSLA 
Hue, Saturation, Lightness, Alpha
HSLA(0, 100%, 50%, 0.5) = Red with 0.5 Transparency 

CSS Background

1. Color : Specifies background color of an element

2. Image: Specifies an image to use as the background of an element

3. Repeat: Repeat an image both horizontally and vertically

4. Attachment: Specifies whether the image should be scroll or be fixed

5. Shorthand: color imageUrl repeat attachment
